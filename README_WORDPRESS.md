# تعليمات استيراد المحتوى إلى ووردبريس (WordPress)

هذا المشروع يحتوي على 2,000 مقال علمي (1,000 في علم النفس و 1,000 في علوم الأعصاب) جاهزة للاستيراد المباشر.

## الملفات المطلوبة
ستجد ملفاً مضغوطاً باسم `wordpress_import_package.zip` في مجلد `public` أو يمكنك تحميله من الموقع بعد نشره.

## خطوات الاستيراد:

1. **فك الضغط**: قم بفك ضغط ملف `wordpress_import_package.zip` للحصول على ملفات XML الأربعة:
   - `psychology_1.xml`
   - `psychology_2.xml`
   - `neuroscience_1.xml`
   - `neuroscience_2.xml`

2. **الدخول إلى ووردبريس**: سجل الدخول إلى لوحة تحكم موقعك (Dashboard).

3. **تثبيت أداة الاستيراد**:
   - اذهب إلى **أدوات (Tools)** -> **استيراد (Import)**.
   - ابحث عن **WordPress** في القائمة واضغط على **التثبيت الآن (Install Now)**.
   - بعد التثبيت، اضغط على **تشغيل المستورد (Run Importer)**.

4. **رفع الملفات**:
   - قم برفع ملفات XML واحداً تلو الآخر.
   - بعد رفع كل ملف، سيطلب منك ووردبريس تعيين مؤلف للمقالات.
   - اختر "إنشاء مستخدم جديد" باسم **juanmoisesd** أو قم بتعيين المقالات لمستخدم موجود.
   - **هام**: تأكد من تفعيل خيار "Download and import file attachments" إذا كنت ترغب في جلب الصور (رغم أن هذا المحتوى نصي بشكل أساسي).

5. **التحقق**:
   - اذهب إلى **مقالات (Posts)** للتأكد من ظهور المقالات الجديدة.
   - ستجد المقالات مصنفة تلقائياً تحت "علم النفس" و "علوم الأعصاب".

## مواصفات المحتوى:
- **المؤلف**: د. خوان مويسيس دي لا سيرنا.
- **التنسيق**: تنسيق علمي (ملخص، مقدمة، منهجية، نتائج، مناقشة، مراجع).
- **الطول**: الحد الأدنى 3,000 كلمة لكل مقال.
- **اللغة**: العربية.

---
**ملاحظة**: بسبب حجم المقالات الكبير (2,000 مقال)، يفضل رفع الملفات في بيئة استضافة مستقرة أو رفعها على دفعات إذا كانت الاستضافة محدودة الموارد.
